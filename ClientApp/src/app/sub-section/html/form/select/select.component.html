HTML-элемент <select> представляет собой элемент управления, предоставляющий меню опций для создания выпадающего списка.

Интерактивный пример

В приведенном выше примере показано типичное использование <select>. Ему присваивается атрибут id, позволяющий связать его с <label> в целях доступности, а также атрибут name для представления имени связанной точки данных, отправленной на сервер. Каждый пункт меню определяется элементом <option>, вложенным в <select>.

Каждый элемент <option> должен иметь атрибут value, содержащий значение данных для отправки на сервер при выборе этого параметра. Если атрибут value не включен, значением по умолчанию является текст, содержащийся внутри элемента. Вы можете включить атрибут selected в элемент <option>, чтобы сделать его выбранным по умолчанию при первой загрузке страницы.

Элемент <select> имеет несколько уникальных атрибутов, которые вы можете использовать для управления им, например, multiple, чтобы указать, можно ли выбрать несколько параметров, и size, чтобы указать, сколько параметров должно отображаться одновременно. Он также принимает большинство общих атрибутов ввода формы, таких как required, disabled, autofocus и т. д.

Вы можете дополнительно вложить элементы <option> в элементы <optgroup>, чтобы создать отдельные группы параметров внутри раскрывающегося списка.

Атрибуты

autocomplete
Строка, предоставляющая подсказку для функции автозаполнения пользовательского агента.
autofocus
Этот логический атрибут позволяет указать, что элемент управления формы должен иметь фокус ввода при загрузке страницы. Только один элемент формы в документе может иметь атрибут автофокуса.
disabled
Этот логический атрибут указывает, что пользователь не может взаимодействовать с элементом управления.
form
Этот атрибут позволяет связать элементы <select> с <form> в любом месте документа, а не только внутри формы. Он также может переопределить элемент-предок <form>.
multiple
Этот логический атрибут указывает, что в списке можно выбрать несколько параметров, большинство браузеров будут отображать окно списка с прокруткой вместо выпадающего списка с одной строкой. Если он не указан, то одновременно может быть выбран только один вариант.
name
Этот атрибут используется для указания имени элемента управления.
required
Логический атрибут, указывающий, что элемент управления должен быть заполнен.
size
Если элемент управления представлен в виде прокручиваемого списка (например, когда указано multiple), этот атрибут представляет количество строк в списке, которые должны быть видны одновременно. Браузеры не обязаны представлять элемент выбора в виде прокручиваемого списка. Значение по умолчанию — 0.

Примечания по использованию

На настольном компьютере существует несколько способов выбора нескольких параметров в элементе <select> с множественным атрибутом.

Пользователи мыши могут удерживать клавиши Ctrl, Command или Shift (в зависимости от того, что имеет смысл для вашей операционной системы), а затем щелкать несколько параметров, чтобы выбрать или отменить их выбор.

Пользователи клавиатуры могут выбирать несколько смежных элементов:

  • Сменить фокус на элементе <select> (например, с помощью Tab).
  • Выбор элемента в верхней или нижней части диапазона, который они хотят выбрать, с помощью клавиш PgUp и PgDn для перехода вверх и вниз по параметрам.
  • Удерживая нажатой клавишу Shift, а затем используя клавиши Up и Down, увеличивайте или уменьшайте диапазон выбранных элементов.

Пользователи клавиатуры могут выбирать несколько несмежных элементов:

  • Сменить фокус на элементе <select> (например, с помощью Tab).
  • Удерживая нажатой клавишу Ctrl, затем с помощью клавиш Up и Down. После сфокусирования на нужный элемент нажать Shift.
  • Нажатие Space для выбора/отмены выбора.
