Существует множество различных типов селекторов CSS, позволяющих применять правила к конкретным элементам НТМL-документа.

Селекторы CSS чувствительны к регистру, поэтому они должны точно соответствовать именам элементов и значениям атрибутов. Существуют более продвинутые селекторы, позволяющие выбирать элементы на основании атрибутов и их значений.

Интерактивный пример

Перечень селекторов

Универсальный селектор

Соответствует любому HTML-элементу. Например, * {{ '{' }} margin:0;{{ '}' }} обнулит внешние отступы для всех элементов сайта. Также селектор может использоваться в комбинации с псевдоклассом или псевдоэлементом: *:after, *:checked.

Селектор типа

Селектор типа в CSS позволяет выбирать все элементы с определенным типом тега, таким как <p>, <h1>, <ul> и т.д.

Селектор класса

Селекторы класса позволяют задавать стили для одного и более элементов с одинаковым именем класса, размещенных в разных местах страницы или на разных страницах сайта.

Селектор идентификатора

Селектор идентификатора - это один из видов селекторов в CSS, который используется для выбора элемента на основе его уникального идентификатора. Идентификатор - это атрибут HTML-элемента, который определяет его уникальность в пределах документа.

Селектор идентификатора в CSS обозначается символом #, за которым следует имя идентификатора. Например, для выбора элемента с идентификатором myDiv, мы можем использовать селектор #myDiv. Как только мы выбрали элемент с помощью селектора идентификатора, мы можем применить к нему набор свойств CSS. \u002E

Селектор идентификатора позволяет создавать уникальные стили для конкретного элемента на странице, что делает его особенным и выделяет на фоне других элементов.

Дочерний селектор

Дочерний элемент является прямым потомком содержащего его элемента и обозначается символом >. У одного элемента может быть несколько дочерних элементов, а родительский элемент у каждого элемента может быть только один. Дочерний селектор позволяет применить стили только если дочерний элемент идёт сразу за родительским элементом и между ними нет других элементов, то есть дочерний элемент больше ни во что не вложен.

В данном примере мы выбрали нумерованный список ol родителем которого является элемент маркированного списка li и применили стили для офомления заднего фона для большей наглядности.

Селектор потомка

Селектор потомка в CSS позволяет выбрать элементы, которые являются потомками другого элемента. Он состоит из двух селекторов, разделенных пробелом. Первый селектор указывает на родительский элемент, а второй на потомка.

Селектор соседнего элемента

Селектор соседнего элемента позволяет выбрать элемент, который следует непосредственно за другим элементом и оба элемента должны быть дочерними элементами одного родительского элемента.

В этом примере мы выбираем элемент p, который следует непосредственно за заголовком h2 и выделяем его. В результате окрашены только первые абзацы после каждого заголовка.

Селектор обобщенных родственных элементов

Соответствует элементу, являющемуся сестринским по отношению к другому элементу, хотя не обязательно расположенному перед ним.

В этом примере мы выбираем элемент p, который следует за заголовком h2 и выделяем его. В результате окрашены все абзацы после каждого определенного заголовка.

Селектор атрибута

Селектор атрибута в CSS - это способ выбора элементов на основе наличия или значения атрибута в HTML-разметке.

[атрибут]
Выбирает элементы, у которых есть указанный атрибут (значение атрибута не имеет значения).
[атрибут ='значение']
Выбирает элементы, у которых указанный атрибут имеет точное значение.
[атрибут ~='значение']
Выбирает элементы, у которых указанный атрибут содержит заданное значение в качестве одного из слов, разделенных пробелами.
[атрибут |='значение']
Выбирает элементы, у которых список значений атрибута начинается с указанного слова.
[атрибут ^='значение']
Выбирает элементы, у которых атрибут начинается с указанного значения.
[атрибут $='значение']
Выбирает элементы, у которых указанный атрибут заканчивается на заданное значение.
[атрибут *='значение']
Выбирает элементы, у которых указанный атрибут содержит заданное значение в любом месте.

Селектор псевдокласса

Псевдоклассы — это классы, фактически не прикрепленные к HTML-элементам. Селектор псевдокласса в CSS - это ключевое слово, которое добавляется к селектору и позволяет выбирать элементы не только на основе их имени тега, класса или идентификатора, но и на основе состояния или свойства элемента.

Существует множество различных псевдоклассов, которые можно использовать в CSS, например:

:active
Применяется к элементу во время активации пользователем (когда пользователь нажимает на элемент левой кнопкой мыши).
:focus
Применяется к элементу, который имеет фокус ввода.
:hover
Применяется к элементу, на который наведен курсор мыши.
:visited
Применяется к элементу после того, как он был посещен пользователем.
:link
Применяется к не посещенным ссылкам.
:empty
Применяется к элементу, который не имеет потомков.
:checked
Применяется к отмеченному элементу формы.
:disabled
Применяется к элементу формы, который отключен.
:enabled
Применяется к элементу формы, который включен.
:target
Применяется к элементу, который соответствует текущему URL-адресу.
:not(selector)
Применяется к элементам, которые не соответствуют заданному селектору.
:in-range
Поля формы, значения которых находятся в заданном диапазоне.
:out-of-range
Поля формы, значения которых не входят в установленный диапазон.
:lang()
Элементы с текстом на указанном языке.

Пример использования селектора псевдокласса

Селектор структурных псевдоклассов

Селектор структурных псевдоклассов - это тип псевдокласса в CSS, который позволяет выбирать элементы на основе их позиции в структуре документа относительно других элементов.

:nth-child(odd)
Нечётные дочерние элементы.
:nth-child(even)
Чётные дочерние элементы.
:nth-child(3n)
Каждый третий элемент среди дочерних.
:nth-child(3n+2)
Выбирает каждый третий элемент, начиная со второго дочернего элемента (+2).
:nth-child(n+2)
Выбирает все элементы, начиная со второго.
:nth-child(3)
Выбирает третий дочерний элемент.
:nth-last-child()
В списке дочерних элементов выбирает элемент с указанным местоположением, аналогично с :nth-child(), но начиная с последнего, в обратную сторону.
:first-child
Позволяет оформить только самый первый дочерний элемент.
:last-child
Позволяет форматировать последний дочерний элемент.
:only-child
Выбирает элемент, являющийся единственным дочерним элементом.
:empty
Выбирает элементы, у которых нет дочерних элементов.
:root
Выбирает элемент, являющийся корневым в документе — элемент <html>.

Пример использования селектор структурных псевдоклассов

Селектор структурных псевдоклассов типа

Указывают на конкретный тип дочернего элемента.

:nth-of-type()
Выбирает элементы по аналогии с :nth-child(), при этом берёт во внимание только тип элемента.
:first-of-type
Выбирает первый дочерний элемент данного типа.
:last-of-type
Выбирает последний элемент данного типа.
:nth-last-of-type()
Выбирает элемент заданного типа в списке элементов в соответствии с указанным местоположением, начиная с конца.
:only-of-type
Выбирает единственный элемент указанного типа среди дочерних элементов родительского элемента.

Пример использования селектора структурного псевдокласса типа

Селектор псевдоэлемента

Псевдоэлементы используются для добавления содержимого, которое генерируется с помощью свойства content

:first-letter
Применяет стили к первой букве указанного элемента.
:first-line
Применяет стили к первой строке указанного элемента.
:before
Вставляет контент до указанного элемента.
:after
Вставляет контент после указанного элемента.
