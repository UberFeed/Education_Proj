Спецификация CSS даёт неограниченные возможности для оформления таблиц. По умолчанию таблица и ячейки таблицы не имеют видимых границ и фона, при этом ячейки внутри таблицы не прилегают вплотную друг к другу.

Ширина ячеек таблицы определяется шириной их содержимого, поэтому ширина столбцов таблицы может быть разной. Высота всех ячеек ряда одинаковая и определяется высотой самой высокой ячейки.

Интерактивный пример

Границы таблицы

Границы таблицы задаются свойством border

Свойство border является особенно полезным, когда требуется задать одинаковые параметры для всех четырех границ элемента. Однако, чтобы сделать границы разными, можно использовать длинные свойства border-width, border-style и border-color, которые принимают различные значения для каждой грани.

Значение Описание
line-width Определяет толщину границы. По умолчанию устанавливается значение medium, если отсутствует явное указание толщины.
line-style Устанавливает стиль границы. По умолчанию отсутствует, то есть граница не отображается.

Принимает следующи значения:

  • none - без границы (значение по умолчанию).
  • hidden - скрытая граница, которая занимает место.
  • dotted - пунктирная граница.
  • dashed - штриховая граница.
  • solid - сплошная граница.
  • double - двойная граница.
  • groove - выдавленная граница. Рисуется двумя цветами — темнее на внешней стороне и светлее на внутренней.
  • ridge - рельефная граница. Рисуется двумя цветами — светлее на внешней стороне и темнее на внутренней.
  • inset - внутренняя граница. Рисуется двумя цветами — темнее на внешней стороне и светлее на внутренней.
  • outset - внешняя граница. Рисуется двумя цветами — светлее на внешней стороне и темнее на внутренней.
color Устанавливает цвет границы. По умолчанию используется текущий цвет, если он отсутствует.

Пример оформления границ таблицы

Заголовок таблицы

Добавить заголовок в таблицу можно с помощью элемента <caption>, а с помощью свойства caption-side его можно поместить перед таблицей или под ней. Для горизонтального выравнивания текста заголовка применяется свойство text-align.

Значение Описание
top Заголовок таблицы располагается над таблицей. Значение по умолчанию.
bottom Располагает заголовок под таблицей.
initial Устанавливает значение свойства в значение по умолчанию.
inherit Наследует значение свойства от родительского элемента.

Пример оформления границ таблицы

Промежуток между рамками таблицы

Рамки ячеек таблицы по умолчанию разделены небольшим промежутком. Если задать для таблицы border-collapse: collapse, то промежуток уберётся.

Значение Описание
separate Рамки ячеек располагаются раздельно.
collapse Рамки ячеек сливаются в одну, а промежутки между рамками убираются.
initial Устанавливает значение свойства в значение по умолчанию.
inherit Наследует значение свойства от родительского элемента.

Пример добавления/удаления промежутка между ячейками

Увеличение промежутка между ячейками таблицы

С помощью свойства border-spacing можно менять расстояние между рамками ячеек. Данное свойство применяется к таблице в целом.

Значение Описание
x-lenght, y-lenght Добавляет промежутки между рамками как по вертикали, так и по горизонтали. Если заданы две длины, то первая всегда определяет горизонтальный промежуток, а вторая — вертикальный.
initial Устанавливает значение свойства в значение по умолчанию.
inherit Наследует значение свойства от родительского элемента.

Пример добавления/удаления промежутка между ячейками

Отображение пустых ячеек таблицы

Свойство empty-cells скрывает или показывает пустые ячейки. Действует только на ячейки, которые не содержат какой-либо контент. Если для ячейки задан фон, а для таблицы задано table {{ '{' }}border-collapse: collapse;{{ '}' }}, то ячейка не будет скрыта.

Значение Описание
show Рамка и фон пустой ячейки будут отрисовываться так же, как для ячейки таблицы, имеющей содержимое.
hide Если все ячейки строки пусты, то вся строка отображается так, если бы было задано значение display: none.
initial Устанавливает значение свойства в значение по умолчанию.
inherit Наследует значение свойства от родительского элемента.
