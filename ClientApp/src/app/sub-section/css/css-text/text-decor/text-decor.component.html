Оформление текста является одной из ключевых задач в веб-дизайне. Он помогает сделать текст более читаемым и привлекательным для пользователей, а также улучшает общее визуальное восприятие сайта.

Интерактивный пример

Создание декораций

Линии верхнего и нижнего подчеркивания, перечеркивание

Свойство text-decoration в CSS определяет оформление текста, включая его подчеркивание, линию через текст, цвет и стиль.

Значение Описание
none Текст без оформления.
underline Добавляет линию под текстом.
overline Добавляет линию над текстом.
line-through Добавляет линию через текст.
blink Делает текст мерцающим (это значение устарело и рекомендуется не использовать его).
inherit Наследует значение свойства от родительского элемента.

Кроме того свойство text-decoration является сокращением для следующих свойств:

  • text-decoration-color - цвет линии.
  • text-decoration-line - тип линии.
  • text-decoration-style - стиль линии.
  • text-decoration-thickness - задает толщину обводки линии декорации, которая используется для текста.

Свойство CSS text-decoration-skip-ink определяет, как рисуются верхние и нижние линии декорации, когда они пересекают засечки и хвосты символов.

Существует также свойсто text-underline-offset не является частью сокращенной записи text-decoration. В то время как у элемента может быть несколько линий text-decoration, text-underline-offset влияет только на подчеркивание и не влияет на другие возможные опции декорации линий, такие как верхнее подчеркивание (overline) или зачеркивание (line-through).

Преобразование текста

Свойство text-transform стилизует текст. Оно не влияет на базовое содержимое и не должно влиять на содержимое операции копирования и вставки простого текста.

Значение Описание
none Значение по умолчанию, означает отсутствие эффектов.
capitalize Изменяет написание первой буквы каждого слова в элементе, делая её прописной.
overline Добавляет линию над текстом.
uppercase Выводит все слова в элементе прописными буквами.
lowercase Выводит все слова в элементе строчными буквами.
initial Устанавливает значение свойства в значение по умолчанию.
inherit Наследует значение свойства от родительского элемента.

Создание тени

Свойство CSS text-shadow добавляет тень к тексту. Оно принимает список теней, разделенных запятыми, которые будут применяться к тексту и любым его декорациям. Каждая тень описывается комбинацией смещений X и Y от элемента, радиусом размытия и цветом.
