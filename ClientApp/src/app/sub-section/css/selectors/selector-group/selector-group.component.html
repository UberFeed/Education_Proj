Комбинация селекторов позволяет выбирать элементы, которые соответствуют нескольким условиям одновременно.

Группировка селекторов в CSS позволяет применять один и тот же набор правил к нескольким элементам. Для этого селекторы перечисляются через запятую.

Интерактивный пример

Примеры комбинации различных селекторов:

Комбинация селекторов позволяет выбирать элементы, которые соответствуют нескольким условиям одновременно. Вот несколько примеров комбинации различных селекторов:

Комбинация селекторов потомка и класса: выбираются элементы, которые являются потомками других элементов и имеют определенный класс. Например, чтобы выбрать все элементы span с классом my-class,которые находятся внутри элементов p, используйте селектор p span.my-class.

Комбинация селекторов псевдоклассов и класса: выбираются элементы, которые имеют определенный класс и определенное состояние, заданное с помощью псевдокласса. Например, чтобы выбрать все элементы button с классом my-class, которые находятся в состоянии hover, используйте селектор button.my-class:hover.

Пример группировки различных селекторов

Комбинация нескольких селекторов через запятую. Выбираются элементы, которые соответствуют любому из перечисленных селекторов. Благодоря этому мы можем применить один и тот же стиль к нескольким элементам.
