Элемент <textarea> используется для создания многострочных областей ввода текста. В отличие от других элементов формы, он не является пустым, а следовательно, для него должны следует указать открывающий и закрывающий теги.

Текст, помещенный между ними, будет выведен в поле при загрузке страницы.

Интерактивный пример

Атрибут cols устанавливает ширину текстовой области (измеряется в количестве символов в одной строке). Атрибут rows задает высоту поля в строках.

Приведенный выше пример демонстрирует ряд возможностей <textarea>:

  • Атрибут id, позволяющий связать <textarea> с элементом <label> в целях доступности.
  • Атрибут name для установки имени связанной точки данных, отправляемой на сервер при отправке формы.
  • rows и cols, чтобы вы могли указать точный размер <textarea>. Их установка является хорошей идеей для согласованности, так как браузеры по умолчанию могут различаться.
  • Содержимое по умолчанию, введенное между открывающим и закрывающим тегами. <textarea> не поддерживает атрибут value.

Элемент <textarea> также принимает несколько атрибутов, общих для форм <input>, таких как autocomplete, autofocus, disabled, placeholder, readonly и required.
